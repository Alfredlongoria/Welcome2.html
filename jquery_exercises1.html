  • The jQuery object We will be using the jQuery object to find and create HTML elements from the DOM. The jQuery object is defined in the API documentation: Return a collection of matched elements either found in the DOM based on passed argument(s) or created by passing an HTML string. In jQuery, we commonly use the dollar sign $, an alias of jQuery to reference the jQuery object, as we will see below.
